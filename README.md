# flask-music-recommender
This Flask project provides an API for recommending songs based on audio features and titles. It incorporates machine learning models to suggest 10 songs similar to the input provided.
Audio Feature Extraction: Extracts audio features from uploaded music files.
Song Recommendation: Utilizes an ML model to recommend 10 similar songs based on audio features and titles.
RESTful API: Provides a RESTful API endpoint to retrieve song recommendations.
Technologies Used:
Flask
Pandas
Scikit-learn
Clone the repository.
Install the required dependencies using pip install -r requirements.txt.
Run the Flask server using python app.py.
Access the application in your browser at http://localhost:5000.
